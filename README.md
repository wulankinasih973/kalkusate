**KalkuSate** adalah aplikasi web berbasis **Laravel 12.34.0** yang dirancang untuk membantu UMKM Warung Sate Suka Mulya Majenang dalam menghitung modal, pendapatan, dan keuntungan secara otomatis. Proyek ini dibuat sebagai bagian dari **Praktik Kerja Lapangan (PKL)** dan berfungsi untuk meningkatkan efisiensi pencatatan transaksi dan laporan keuangan.

---

## Fitur Utama

- Kalkulator keuntungan: menghitung keuntungan dari transaksi penjualan.
- CRUD laporan transaksi: menyimpan, menampilkan, dan menghapus data transaksi.
- Ekspor laporan ke PDF & CSV.
- Filter laporan berdasarkan **range tanggal** sebelum ekspor.
- Halaman **About** berisi profil UMKM dan profil pengembang.
- Tampilan web responsif untuk desktop dan mobile.

---

## Teknologi yang Digunakan

- **PHP 8.2.12** dan **Laravel 12.34.0**
- **MySQL** untuk database
- **HTML, CSS, Bootstrap**
- **JavaScript** untuk interaksi front-end
